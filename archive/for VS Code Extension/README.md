## 项目一句话定义

**CodeReader 是一个 IDE 形态的 AI 代码阅读器，通过持久化的行级解释层，帮助使用 AI 编程的人逐行、逐块、逐文件理解并掌握自己的代码。**

## 核心原则

- **代码优先**：不是问答聊天框，而是以代码文件为主体的阅读界面。
- **行级映射**：解释必须稳定对应到代码行、代码块、函数、文件与模块。
- **解释持久化**：同一文件再次打开时，解释信息应可直接回看，而不是每次随机重生成。
- **认知沉淀**：CodeReader 不只是“解释代码”，而是帮助用户建立对项目的长期认知资产。
- **风险可见**：帮助用户识别 AI 生成代码中的空值、异常、安全、性能、依赖、架构与业务风险。
